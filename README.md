## 👋 About Me

I build production-grade AI systems — from agentic operations assistants with governance controls to document automation platforms and full-stack web applications. My work spans LLM orchestration, multi-provider fallback reasoning, and backend APIs backed by PostgreSQL and deployed on AWS. Every system I ship includes audit trails, safety guardrails, and self-healing capabilities.
